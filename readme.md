Student Courses Tracker Management
===================================
This simple PHP based website helps students of
East West University
to keep track of completed and on goinging courses
along side with its schedule, section and class room number.

Requirements
============
* PHP >=5.3
* MySql >=5.5

Installation
============
* First modify the file @ db/db.php according to your mySql configuration.
* Also determine the database you're using.
* Use the "sql.txt" file to add the tables and some sample data.
* Rest is up to you.
